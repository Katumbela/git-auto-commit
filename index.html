Git Commit CLI

A simple command-line tool to automate the process of adding and committing files in a Git repository.

Features

  • Add and commit untracked files
  • Add and commit modified files
  • Add and commit all files

Installation

You can install the package globally using npm:

npm install -g git-commit-lib

Usage

After installing the package, you can use the following commands:

  • npx commit-kat add-untracked - Add and commit untracked files
  • npx commit-kat add-modified - Add and commit modified files
  • npx commit-kat add-and-commit - Add and commit both untracked and modified files

Examples

Adding and Committing Untracked Files:

npx commit-kat add-untracked

Adding and Committing Modified Files:

npx commit-kat add-modified

Adding and Committing All Files:

npx commit-kat add-and-commit
